# goBIGai Portfolio

A neo-brutalist, single-page portfolio for **Anirudh Voruganti** (goBIGai) built with Next.js 15, Tailwind CSS, and lucide-react icons. The site highlights GenAI offerings, case studies, the technology arsenal, and an easy CTA so it can be deployed to [Vercel](https://vercel.com/) with no additional configuration.
